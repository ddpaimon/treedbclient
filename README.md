# Treedb client

## Requirements:
Python 3.6 <br />
requests 2.18.4

## Launch:
```
git clone https://github.com/ddpaimon/treedbclient.git
cd treedbclient
sudo pip install -r requirements.txt
python start.py
```

## Description:
testdb.json file contains base database view, and applied after reset.
